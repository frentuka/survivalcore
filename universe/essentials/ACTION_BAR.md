# ActionBar Essential Module (ActionBarEssential)

## 📌 Code Path
* **Directory:** `src/main/kotlin/site/ftka/survivalcore/essentials/actionbar/`

## 🎯 Main Purpose
Provides a clean, streamlined facade for sending dynamic text action bar messages directly to players, controlling display times and formats cleanly.

## 🔗 Connections & Dependencies
* **Utility Facade:** Extends high-level APIs for Apps and Services to deliver quick, non-intrusive status micro-messages without spamming the player's primary chat window.\n
